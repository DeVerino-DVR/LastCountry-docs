Events

Convention: lo:<domain>:<action>. The lo:Server:* events are mirrored as RSGCore:Server:* by the bridge (interop with RSG resources without rsg-core).

Direction: C→S client to server · S→C server to client · local same side.

Lifecycle​

EventDirectionDescription
lo:session:createdlocal (S)Session built (userId/charId/adminLevel/vip statebags set).
lo:Server:PlayerLoadedlocal (S)All player/character systems initialized.
lo:Server:OnPlayerLoadedlocal (S)Player connected (once).
lo:Server:OnPlayerUnloadlocal (S)Player disconnecting.
lo:Client:OnPlayerLoadedS→CCharacter loaded and spawned on the client.
lo:Client:OnPlayerUnloadS→CCharacter disconnecting on the client.
-- client: react to load (prefer Core.OnPlayerLoaded)
AddEventHandler('lo:Client:OnPlayerLoaded', function() end)

Player mutations (relayed as RSGCore:Server:*)​

EventDirectionDescription
lo:Server:OnMoneyChangelocal (S)(src, type, amount, action, reason).
lo:Server:OnJobUpdatelocal (S)Job changed.
lo:Server:OnGangUpdatelocal (S)Gang changed.
lo:Server:SetMetaDatalocal (S)Metadata key set.
lo:Player:SetPlayerDataS→CFull PlayerData sync.
lo:Client:OnMoneyChange / OnJobUpdate / OnGangUpdateS→CSync the mutation to the client.

Characters​

EventDirectionDescription
lo:server:getCharactersC→SList of the account's characters.
lo:server:getCharacterC→SLoad a character.
lo:server:switchCharacterC→SSwitch character (by id).
lo:server:setFavoriteCharacterC→SMark a character as favorite.
lo:server:createCharacterC→SCreate a character (minimal).
lo:server:createCharacterWithSkinC→SCreate with skin + outfit (creator).
lo:server:saveCharacterC→SSave appearance + outfit.
lo:server:deleteCharacterC→SDelete a character.
lo:client:receiveCharacter(s)S→CReturn the loaded character / the list.
lo:client:noCharacterS→CNo character: offer creation.
lo:client:logoutS→CBack to selection.
lo:client:openCharacters / characterFlowEndedlocal (C)Open selection / end of the creator flow.

creator module​

EventDirectionDescription
lo:creator:start / editS→COpen the creator (creation / editing /skin).
lo:creator:enterBucket / exitBucketC→SRouting bucket isolation during creation.
lo:creator:toggleBlacklist (key)C→SHide/restore an element (staff ≥ 30) → broadcast blacklistUpdated.
lo:creator:setCategory (key, cat)C→SRecategorize (staff ≥ 30) → broadcast overridesUpdated.
lo:creator:extractSave (cat, content)C→S latentWrite an extraction file to modules/creator/client/data/.
lo:creator:blacklistUpdated / overridesUpdatedS→C broadcastStaff state resynchronized.

Inventory (useful excerpts)​

EventDirectionDescription
lo:inventory:usedItem (slot)C→SUse an item.
lo:inventory:swapItemsC→SMove/swap between slots/inventories.
lo:inventory:giveItem (targetId, slot, count)C→SGive an item to a player.
lo:inventory:updateSlots / refreshAllS→CPush slot changes / full refresh.
lo:inventory:itemNotifyS→CItem action toast.
lo:inventory:createDrop / removeDropS→CGround drop created / removed.
lo:Server:UsedItemlocal (S)Item used (all listeners).

World / editor​

EventDirectionDescription
lo:world:syncS→CSend all entities to the client.
lo:world:add / update / remove / resetS→C broadcastEntity created / modified / removed / reload.
lo:world:editorAdd / editorRemoveC→S (staff)Add / remove via the panel.
lo:editor:apply / bulkS→C broadcastApply a Config change (single / batch).
lo:ped:brokenlocal (C)Streaming of a ped failed (model unavailable).

Other domains​

EventDirectionDescription
lo:client:notify (msg, kind, duration)S→CToast (used by server-side Core.Notify).
lo:client:DrawText / ChangeText / HideTextS→CText overlay.
lo:hud:setStatus / setMinimap / setState / setVisibleS→CHUD control.
lo:jobs:syncS→CPush job config.
lo:paycheck:paidS→CPaycheck notification.
lo:voice:syncCallDataS→CSync call members/data.
