Skin — Appearance client

The appearance API (core/client/character/skin.lua). It accepts both shop items (CLOTHING_ITEM_* hashes) and meta-tag assets (extracted from peds). All functions take an optional target ped (default: the player ped).

The SkinAsset table: { hash?, drawable?, albedo?, normal?, material?, palette?, tint0?, tint1?, tint2? }.

Skin.ApplySkin​

Skin.ApplySkin(skin, target?)

Apply a full skin (jo.component.applySkin format + lo extensions: headHash, bodyUpperHash/bodyLowerHash, hideUpperBody/hideLowerBody, whistle, eyesIndex/eyesTint/eyesPalette).

Parameters​

NameTypeDescription
skintableFull skin table.
target?integerTarget ped (default player).

Returns​

nil

Example​

Skin.ApplySkin(Core.GetPlayerData().skin)

Skin.ApplyClothes​

Skin.ApplyClothes(clothes, target?)

Apply a clothes table.

Parameters​

NameTypeDescription
clothestable<string, SkinAsset>Category → asset.
target?integerTarget ped.

Returns​

nil

Example​

Skin.ApplyClothes({ hats = { hash = `CLOTHING_ITEM_M_HAT_001` } })

Skin.Apply​

Skin.Apply(category, data, target?)

Apply one component (shop item hash or meta-tag asset).

Returns​

Example​
