Core — Camera client

A reusable scripted camera (only one active at a time): orbit, fixed point, interpolated zoom, ray-picking.

The CoreCameraOpts table:

FieldTypeDescription
coordsvector3Initial camera position.
fov?numberField of view (clamp 20..120, default = gameplay cam).
point?vector3Fixed point to look at (ignored if orbit is given).
orbit?CoreCameraOrbitOrbit around a point (followed each frame).
interp?integerRenderScriptCams fade duration (ms, default 600).

The CoreCameraOrbit table: { focus: fun(): vector3|nil, dist, yaw, pitch }.

Core.Camera.Create​

Core.Camera.Create(opts)

Create and activate the camera.

Parameters​

NameTypeDescription
optsCoreCameraOptsCamera options.

Returns​

integer | nil — the camera handle.

Example​

Core.Camera.Create({
    coords = vector3(100.0, 200.0, 32.0),
    point = vector3(100.0, 205.0, 31.0),
    fov = 50.0,
    interp = 800,
})

Core.Camera.SetOrbit​

Core.Camera.SetOrbit(focus, dist, yaw, pitch)

(Re)define the orbit.

NameTypeDescription
focusfun(): vector3|nilLooked-at point (recomputed each frame).
distnumberDistance to the point.
yawnumberHorizontal angle (radians).
pitchnumberVertical angle (radians).

Returns nil.

Core.Camera.GetOrbit​

Core.Camera.GetOrbit()

Returns { dist, yaw, pitch } or nil.

Core.Camera.GetHandle()

The native camera handle, or nil.

Core.Camera.GetCoord​

Core.Camera.GetCoord()

The current camera position (vector3), or nil.

Core.Camera.SetFov(value)

Set the FOV (clamped 20..120).

NameTypeDescription
valuenumberNew FOV.

Core.Camera.GetFov()

The current FOV, or nil.

Core.Camera.InterpToPoint​

Core.Camera.InterpToPoint(point, dist, zoomFov, duration?)

Interpolated zoom toward a point.

NameTypeDescription
pointvector3Target point.
distnumberDistance to the point.
zoomFovnumberTarget FOV.
duration?integerInterpolation duration (ms).

Returns nil.

Core.Camera.InterpToOrbit​

Core.Camera.InterpToOrbit(duration?)

Interpolate back to the orbital view.

NameTypeDescription

Core.Camera.IsInterpolating()

Core.Camera.CursorRay​

Core.Camera.CursorRay(cursorX, cursorY)

Ray-picking: returns the camera position and the ray direction for a cursor position.

NameTypeDescription
cursorXnumberCursor X (0-1).
cursorYnumberCursor Y (0-1).

Returns vector3 | nil, vector3 | nil — position + direction.

Core.Camera.Destroy​

Core.Camera.Destroy(interp?)

Destroy the camera and return control to gameplay.

NameTypeDescription
interp?integerFade-out duration (ms).

Returns nil.

Example​

Core.Camera.Destroy(600)
