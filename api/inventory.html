Core — Inventory server

The inventory module API. The id argument is a player's source, or a stash / drop identifier (string). Item names come from data/items.

Core.Inventory.AddItem​

Core.Inventory.AddItem(id, name, count?, metadata?, slot?)

Add an item to an inventory.

Parameters​

NameTypeDescription
idinteger | stringPlayer source or stash/drop id.
namestringItem name (data/items).
count?integerQuantity (default 1).
metadata?tableItem metadata.
slot?integerTarget slot.

Returns​

boolean

Example​

Core.Inventory.AddItem(src, 'water', 2)

Core.Inventory.RemoveItem​

Core.Inventory.RemoveItem(id, name, count?, metadata?, slot?)

Remove an item from an inventory.

Parameters​

NameTypeDescription
idinteger | stringPlayer source or stash/drop id.
namestringItem name.
count?integerQuantity (default 1).
metadata?tableMatch this metadata.
slot?integerSpecific slot.

Returns​

boolean

Example​

if Core.Inventory.RemoveItem(src, 'water', 1) then
    Core.Hud.AddThirst(20)
end

Core.Inventory.HasItem​

Core.Inventory.HasItem(id, items, amount?)

Does the inventory hold the item(s)?

Parameters​

NameTypeDescription
idinteger | stringPlayer source or stash/drop id.
itemsstring | string[]One name or a list.
amount?integerMinimum amount.

Returns​

boolean

Example​

if Core.Inventory.HasItem(src, { 'lockpick', 'screwdriver' }) then
    -- ...
end

Core.Inventory.GetItemCount(id, name, metadata?)

How many of an item the inventory holds.

NameTypeDescription

namestringItem name.

Returns​

integer

Example​

local n = Core.Inventory.GetItemCount(src, 'gold_bar')

Core.Inventory.CanCarryItem​

Core.Inventory.CanCarryItem(id, name, count, metadata?)

Can the inventory fit this item (weight / slots)?

Parameters​

NameTypeDescription
idinteger | stringPlayer source or stash/drop id.
namestringItem name.
countintegerQuantity to test.
metadata?tableItem metadata.

Returns​

boolean

Example​

if Core.Inventory.CanCarryItem(src, 'crate', 1) then
    Core.Inventory.AddItem(src, 'crate', 1)
end

Core.Inventory.Open​

Core.Inventory.Open(invType, data)

Open an inventory for a player.

Parameters​

NameTypeDescription
invTypestringplayer | stash | shop | crafting | drop …
dataCoreInventoryOpenData{ id?, title?, slots?, maxWeight?, items? }.

Returns​

nil

Example​

Core.Inventory.Open('stash', {
    id = 'house_42_stash',
    title = 'House Stash',
    slots = 50,
    maxWeight = 100000,
})

Core.Inventory.CreateUseableItem​

Core.Inventory.CreateUseableItem(name, callback)

Register an item-use callback (equivalent to CreateUseableItem in RSG).

Parameters​

NameTypeDescription
namestringItem name.
callbackfun(source: integer, item: table)Runs when the item is used.

Returns​

nil

Example​

Core.Inventory.CreateUseableItem('bandage', function(src, item)
    if Core.Inventory.RemoveItem(src, 'bandage', 1) then
        -- heal logic
    end
end)

Core.Inventory.SetMetadata​

Core.Inventory.SetMetadata(id, slot, metadata)

Overwrite a slot's metadata.

Parameters​

NameTypeDescription
idinteger | stringPlayer source or stash/drop id.
slotintegerSlot index.
metadatatableNew metadata.

Returns​

nil

Example​

Core.Inventory.SetMetadata(src, 3, { durability = 80 })

Core.Inventory.Search​

Core.Inventory.Search(id, mode, names, metadata?)

Search an inventory.

Parameters​

NameTypeDescription
idinteger | stringPlayer source or stash/drop id.
mode'slots' | 'count'Return matching slots or a count.
namesstring | string[]Item name(s).
metadata?tableMatch this metadata.

Returns​

table | integer — slots table ('slots') or a count ('count').

Example​

local slots = Core.Inventory.Search(src, 'slots', 'water')
local total = Core.Inventory.Search(src, 'count', 'water')

Core.Inventory.GetItemList​

Core.Inventory.GetItemList()

All item definitions (data/items), indexed by name.

Returns​

table<string, table>

Example​

local items = Core.Inventory.GetItemList()
print(items['water'].label)
