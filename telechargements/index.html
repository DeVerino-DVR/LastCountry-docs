Téléchargements ​

Tout ce qu'il faut pour développer et héberger sur RedM (RDR3). Les liens marqués officiel pointent vers les sources amont.

Client & serveur ​

Client RedM officiel ​

Le client RedM (la « partie RDR2 » de Cfx.re). À installer pour rejouer en multijoueur RedM.

  • Site officiel : redm.net
  • Documentation : docs.fivem.net/docs

Artifacts serveur — FXServer (Windows) officiel ​

Les builds du serveur FXServer (identiques pour FiveM et RedM — un serveur RedM tourne sur FXServer avec une ressource game rdr3). Prends un build master récent et stable.

  • Windows (master) : runtime.fivem.net/artifacts/fivem/build_server_windows/master
  • Linux (master) : runtime.fivem.net/artifacts/fivem/build_proot_linux/master

Quel build prendre ?

Choisis un build recommended/optional stable, pas forcément le tout dernier. Note le numéro du build dans ton serveur : il conditionne les natives disponibles.

Ressources de recherche RDR3 ​

rdr3_discoveries — femga ​

La référence communautaire des snippets RDR3 : des centaines d'exemples de natives, d'animations, de scénarios, de props, classés par thème.

  • Repo : github.com/femga/rdr3_discoveries
  • Téléchargement direct : main.zip

Scripts décompilés RDR2 — outsider31000 ​

Les scripts solo de RDR2 décompilés — la source la plus fiable pour comprendre comment Rockstar enchaîne ses natives.

  • Repo : github.com/outsider31000/rdr2-scripts-decompiled
  • Variante (Halen84) : RDR3-Decompiled-Scripts

RDR3 Native Flags & Enums — Halen84 ​

Les flags et enums attendus par les natives (valeurs de paramètres, masques de bits) — indispensable quand une native prend un int flags ou un enum.

  • Repo : github.com/Halen84/RDR3-Native-Flags-And-Enums
